Spell Check

Descrição

O atributo Spell Check (Em codigo spellcheck), serve fazer uma verificação ortográfica no HTML

* Definir o idioma correto no HTML vai ser fundamental para esse atributo funcionar corretamente
* Este atributo não vai verificar textos que são escritos no código, apenas os textos inseridos pelo usuário

Exemplo

Codigo

<input type="text" name="a" id="1" spellcheck="true" value="Input Text" />
<p contenteditable="true" spellcheck="true">
 <a href="conteteditable.html">Contenteditable</a>
</p>
