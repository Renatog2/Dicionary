Data-*

Descrição

O atributo Data-* (Em codigo data-*), serve para guardar algum dado que possa ser utilizado posteriormente mas não fique visível ao usuário

* Ao declarar um Data-*, não se pode utilizar letras maiusculas sendo necessário fazer a separação de palavras utilizando o -
* A principal ideia do Data-* é trazer informações que possam ser utilizadas no Script da pagina
* É possível utilizar mais de um Data-* por elemento

Exemplo

Codigo

<script>
 function showDetails(animal) {
  let animalType = animal.getAttribute("data-animal-type")
  alert(animal.innerHTML + " é um " + animalType);
 }
</script>

<ul>
 <li onclick="showDetails(this)" data-animal-type="Humano">Renato</li>
 <li onclick="showDetails(this)" data-animal-type="Humano">Gustavo</li>
 <li onclick="showDetails(this)" data-animal-type="Cão">Rex</li>
</ul>
