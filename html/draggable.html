Draggable

Descrição

O atributo Draggable (Em codigo draggable), serve para definir se um elemento pode ser arrastado (Movido) na pagina.

* Aplicação pratica é a exemplo no Google Drive, quando adicionamos um arquivo e podemos arrastalo (draggable) dentro do proprio Drive para outro local

Exemplo

Codigo

<style>
 #quadrado {
  width: 100px;
  height: 100px;
  background-color: grey;
 }
</style>

<div id="quadrado" draggable="false"></div>
<div id="quadrado" draggable="true"></div>
