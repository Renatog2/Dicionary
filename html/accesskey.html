Acess Key

Descrição

Acess Key (Em codigo accesskey) é o atributo que permite utilizar uma tecla de atalho para acessar diretamente um determinado local da pagina HTML. Por exemplo você apertar a tecla Alt + H com uma Acess Key definida no botão "Home", irá ter o mesmo efeito que clicar no botão.

* É altamente recomendado não utilizar Acess Key pois o mesmo depende do idioma (Certas letras não existem em certos idiomas) e a possibilidade de conflito com configurações padrões do navegador (Por exemplo o F1 que é o Ajuda).
* Se realmente necessário o uso, a recomendação é utilizar uma tecla que existe no tipo de teclado "Qwerty" (ç não existe lá por exemplo)

Exemplo

Codigo

<a href="https://www.google.com.br/" accesskey="h">Pressione o atalho + H</a>
<button accesskey="s">Pressione o atalho + S</button>

Implementado

Navegadores "Edge", "IE", "Chrome", "Safari" ou "Opera 15+": Pressione ALT + h
Navegador "Opera" anterior a versão 15: Pressione SHIFT + ESC + h
Navegador "Firefox": Pressione ALT + SHIFT + h
