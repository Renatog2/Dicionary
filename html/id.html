ID

Descrição

O atributo ID (Em codigo id), serve para tornar um elemento unico no seu HTML seja para utilização no CSS ou JavaScript

* Embora o HTML aceite vários elementos com o mesmo ID, é recomendado sempre deixar o ID unico

Exemplo

Codigo

<style>
 .caixa {
  width: 100px;
  height: 100px;
  border: solid 1px red;
 }
 #c1 {
  background-color: greenyellow;
 }
 #c2 {
  background-color: lightcyan;
 }
</style>

<div id="c1" class="caixa"></div>
<br />
<div id="c2" class="caixa"></div>

Implementado

Observe que uma parte da formatação veio da Class e outra do ID, usando justamente o ID para tornar a cor do elemento unico
