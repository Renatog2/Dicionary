Style

Descrição

O atributo Style (Em codigo style), serve para definir o estilo de um objeto.

* Embora de para utilizar o style dentro de uma tag ou diretamente no HTML o recomendado é criar um arquivo CSS para tratar o estilo das paginas

Exemplo

Codigo

<style>
 .ab {
  color: red;
 }
</style>

<p style="color: gray">Exemplo de paragrafo com Style direto na tag</p>
<p class="ab">Exemplo de paragrafo com Style no HTML</p>
