Hidden

Descrição

O atributo Hidden (Em codigo hidden), serve para definir se um atributo será ou não exibido para o usuário final no HTML

* Quando Hidden é utilizado ele automaticamente adiciona no CSS o atributo "display: none;"

Exemplo

Codigo

<p>Você consegue me ver</p>
<p hidden>Você não consegue me ver</p>

Implementado

Você consegue me ver

Você não consegue me ver
