Lang

Descrição

O atributo Lang (Em codigo lang), serve para definir o idioma da pagina HTML que está sendo exibida

* O atributo Lang pode ser declarado em outras tags em qualquer idioma, não sendo necessáriamente no mesmo declarado na tag HTML
* Declarar corretamente este atributo faz a tradução pelo navegador funcionar de maneira mais eficiente, bem como facilita para a leitura por maquina

Exemplo

Codigo

<p lang="pt-BR">Olá Mundo!</p>
<p lang="es">¡Hola mundo!</p>

Implementado

Mande o navegador traduzir esta página, note que o idioma detectado é o Inglês declarado na tag HTML mesmo a maioria do texto sendo pt-BR:

Olá Mundo!

¡Hola mundo!
