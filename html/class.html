Class

Descrição

O atributo Class (Em codigo class), serve para especificar um ou mais nomes para elementos

* É considerado uma boa pratica colocar nomes que tenham ligação com o proposito da Class.

* Diferente do ID que tem recomendação de ser utilizado 1-1, o Class pode ser tanto 1-1 quanto 1-*

Exemplo

Codigo

<style> .exemplo { color: red; } </style>
<h4 class="exemplo">Header 4</h4>
<button class="exemplo">Botão</button>

Implementado

Header 4
